# swipe-it

Example of app that demonstrates swipe to delete feature. No library used!!

## Original Tutorial Link
[Drag and Swipe with Recycler View by Paul Burke](https://medium.com/@ipaulpro/drag-and-swipe-with-recyclerview-b9456d2b1aaf?swoff=true#.yrajxbiyk)

## Technologies Used
[RecyclerView](https://developer.android.com/reference/android/support/v7/widget/RecyclerView.html)

[ItemTouchHelper](https://developer.android.com/reference/android/support/v7/widget/helper/ItemTouchHelper.html)

[ItemTouchHelper.Callback](https://developer.android.com/reference/android/support/v7/widget/helper/ItemTouchHelper.Callback.html)

[Fragments](https://developer.android.com/guide/components/fragments.html)
